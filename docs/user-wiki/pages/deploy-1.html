Шаг 1: Подготовка и создание сервера на Hetzner

  1. Вход в Hetzner Cloud Console
    Инструкция: Откройте Hetzner Cloud Console и войдите в аккаунт.
    Статус: Выполнен.
    Комментарии/Проблемы: Ссылка на дашборд проекта
  2. Создание нового сервера
    Инструкция: В консоли Hetzner нажмите "Add Server" или "+".
    Статус: Выполнен.
    Комментарии/Проблемы: Страница создания сервера
  3. Выбор местоположения
    Инструкция: Выберите локацию (например, Helsinki).
    Статус: Выполнен.
    Выбранное местоположение: Helsinki
  4. Выбор образа ОС
    Инструкция: Выберите Ubuntu 24.04.
    Статус: Выполнен.
    Выбранный образ: Ubuntu 24.04
  5. Выбор типа сервера
    Инструкция: Выберите CX22 или аналогичный.
    Статус: Выполнен.
    Выбранный тип: CX22 (2 vCPU, 4GB RAM, 40GB SSD)
  6. Добавление SSH-ключа
    Инструкция: Добавьте публичный SSH-ключ.
    Статус: Выполнен.
    Действие: Ключ сгенерирован на Windows и добавлен в Hetzner.
    Пошаговая инструкция:
    1. Откройте PowerShell или CMD.
    2. Выполните: ssh-keygen -t ed25519 -C "your_email@example.com"
    3. Примите путь по умолчанию, задайте пароль (или оставьте пустым).
    4. Скопируйте содержимое C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\.ssh\id_ed25519.pub и вставьте в Hetzner.
  7. Настройка сети, томов, брандмауэров, бэкапов
    Инструкция: Оставьте по умолчанию.
    Статус: Выполнен.
  8. Имя сервера и завершение создания
    Инструкция: Придумайте имя, проверьте конфигурацию и создайте сервер.
    Статус: Выполнен.
    Имя сервера: studio-app-server
    IP-адрес: 157.180.87.32
